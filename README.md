# Calamity
A simple data bot that logs messages and use info.

Currently logs
- All messages
- Messages from specific user
- Information of each specific user:
  - Avatar
  - Nick
  - Username
  - Creation Date


# Setup

- Add your own token in main.py
- Create the following folders
  - `misc_info`
  - `user_info`
  - `user_messages`
- Start the bot and let it run
